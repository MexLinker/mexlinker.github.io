ps？

  1. service docker status
  2. wget -qO- http://get.docker.com/ | sh
  3. play with docker – 无敌！
  4. 单词：instance
  5. ps -elf
    -e 所有程序
    -l 采用详细的格式来显示程序状况
    -f 显示UID,PPIP,C与STIME栏位
  6. 容器如果不运行任何进程，则无法存在
  7. -p 将docker主机端口映射到容器内
  8. docker inspect imgID
    中，cmd 展示容器将会执行的命令
  9. sh -c XXX ( 命令从-c后的字符串读取)
    eg: sh -c “ cd /src && node ./app/js
