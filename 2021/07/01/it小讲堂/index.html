  1. GitHub copilot, the openAI’s code tool
  2. cloud native ———— 微服务、容器、云原生、Kubernetes、SOA、PaaS平台、Devops 之间的关系
    1. ﻿k8s __ 基于docker的分布式应用也是如此，区别只是网络虚拟化了、CPU和内存资源也虚拟化了。
    2. PaaS(IaaS、PaaS、SaaS，分布是Infrastructure、Platform、Software as a service)
      Platform指的是软件平台，是应用软件运行的基础平台。
    3. 微服务是可以独立部署的、小的、自治的业务组件，业务组件彼此之间通过消息进行交互。微服务的组件可以按需独立伸缩，具备容错和故障恢复能力。
    4. 云原生
      简单直白的讲，基于CNCF云原生技术开发的应用，能够在业界各个平台里畅行无阻，部署在私有云、公有云里都是一样的技术体系和架构。
      云原生的三大特征是：docker、kubernetes和微服务。此外，云原生强调自动化以提升能够开发效率和运维效率。
    5. DevOps是Development和Operations的组合.
      自动化流程来使得软件构建、测试、发布更加迅速和可靠。
