日长日长兮

继续昨天的Gossip，这次叫我从宝石开始，宝玉石鉴赏，将会是我学的一门课。
首先请不要相信所谓宝石能量、利于健康、加强精神…如果真的要这样思考，不妨想想为什么——比如矿物成分摩擦掉落，被皮肤吸收…不要一股脑想着——去收藏转手，因为也许连它是什么你都搞不清楚。
我想要的态度是——学习的态度，谦虚、虔诚
这其实也是我的人生态度，它很科学、很人文，但是有着明显的缺点

猫眼石！是我先要讲的一种石头，四大宝石最后一个！宝石漂亮吗？其实当你投入其中，你会看到每个宝石独特的光彩。那天晚上是实在睡不着了，就打开抖音，浏览各种宝石。谁喜欢宝石呢？我总是想象，有一位特化的心爱的女人，穿戴者珠宝的场景，的确，一开始给我这个感觉的是她耳朵或者哪里的珍珠。
我该怎么去形容呢？圣洁、高贵…哈哈哈，是给人一种那样的感觉。我还在想，是不是到时候，有一定经济能力情况下会去玩这些

嗯——和XZH聊天，谈起人生态度，我是这样讲的：我说我是一个隐忍、不争不抢的人，不想要句句都争得那么细，有一点得过且过，然后运营着自己精致、而又广泛的小世界…你看我该怎么讲呢？

刚刚，在系狂汗啊领导讲完话后，由于疫情得发展，有可能回不了家，回长沙也很麻烦，麻烦的是——我的车游学不了咯，还有我想找家里那个环境换个心情，做不到了
其实我想到了一点，比如“α毒株”是否具有强大的致死性，可否忽视…

日常（网名）

真的好久没有写一些东西了，现在我来讲讲自己

第一个topic当然是画作，

刚刚削练习削铅笔，拿了一只一次性筷子，把他的一头削得很尖，圆锥的夹角也很小，十分精美，我说“我在练习削铅笔”，这支笔现在还在我的手里，我在思考如何把它运用到画作里，我想到蘸取墨汁，或者在对已经画好的画面上改动，制作模糊与光影的效果。肖
铅笔的确给我很多乐趣，但是在我购买一把美工刀的之前，我还是犹豫不决。第一支铅笔削的有点失败，削笔芯的时候断过一次

我还思考过，我现在使用的、广泛使用的——传统素描工具的优略——zhihu上说，当你完全理解、包容这个工具的局限之后，你才能在这个工具上有所作为——这个工具就是传统白纸与石墨制笔芯木制外壳的铅笔。我想到了很多点，我想，当眼界更高时，会准确的抓住其中一两个点。
局限比如

  1. 只能表现色彩的明度
  2. 天！！！！我竟然找不到缺点！

优越比如

  1. 可以简单地控（创造、擦除）制线条、笔触——结合素描的体积比表现方式与明暗画法，以至于可以在平面上表现几乎任何内容
  2. 工具 的 简单可得、方便性、易掌握性
  3. 前人很多创造、总结的技法
  4. 独立于中性笔类的独特性——阅读前沐浴洗手——仪式感

zhihu上讲，要不断尝试新的绘画工具，去思考这一件事的确很有趣，比如木炭条、钢笔、（我学筷子也是这个想法，可以发现没有什么用）、当然可以试试——水彩笔、彩铅、人血(。。。？)、人灰（、。。。）、油画棒！、墨汁、卡纸。我还在想练习可以用很多方式很多较多，比如说，同样是人体速写，我可以画5min的精微、或者60s的动态，都要练习练习！

讲起这次为什么突然心血来潮地画画，源自于在电视塔对面的矿渣山上，老师的一句“这里我们画一张素描图”(虽然事实上只用简单地标注大致走向与岩性…)，站在那座让人印象深刻的矿渣山上，望着对面的电视塔，我精确地找到了每一个点，精致而又漂亮。惊叹于自己地创作，有尝试去画一辆巴车…
回来后我又画了各种各样的东西，吸引我的一点是，看到自己在进步，画出了让自己感到惊叹的东西。

话说“对画画的来说，最基础的安全感就是可以把形打得准”，这句话很好的表达了我最近的心声。论起“形打得准”，我想这不是灵光乍现，而是长久的日常生活中，潜意识中对事物的观察，对自己眼光的确认，这种缓慢的成长发生于无形之中。
的确最近画画，唯有一点让我引以为豪的就是“形打得准”…

还曾经看过知乎上讲述透视原理，均是云里雾里，知道自己思考了一段时间——提出这样的假设（我没有思考过假设如何验证，而是直接把他当作真理）—— 人眼看到的是扭曲的世界，所谓“扭曲”，是不真实的样子，要论真实地“看到”世界是怎样——我曾想过，是捕捉所有频段地电磁波，或者是好多维度的世界…
是——物体在一个比他小很多的球面上的投影经过变换拉直后的效果
投影便是光线经过直线到达球面的过程，而真正造成所谓“进大远小”、“两点、三点透视”，便是投影对物体坐标从三维道二位球面造成的变换效果
所谓“变换拉直”，却是在人脑中发生的思维过程，（我想到平铺的细胞，似乎又不是..）
总的来说我把这件事情讲清楚了，但是如果放到具体的圆柱，又会有所不清楚，所以重要的是思考

思考的重要——所谓“忘记学到的模式、方法、定式，而去表现自己眼中所看到的世界，培养准确地表现自己眼中世界的能力，而不是现实来套入模式的方式来表达”，我是说思考！
画画不思考——怎么会有所进步？当然我发现当你刚看是做一件事情的时候，唤醒水平会很高，的确就会多很多思考，而之后呢——也许人们就会很容易、自然地掉入机械、无脑的练习

画画的乐趣——刚刚想要尝试表达，但是却又找不到，我先想到一种简单地，操作工具、表现世界的乐趣。但是当然有很多很多其他的乐趣——（这里topic会有所不同，而且讲的深）比如:画自己心中的世界，那将会是一种表达欲，创造欲、有一种灵魂的渴望；而我现在在做的很基础的练习，便是做这件事情的工具，当然自动铅笔也可以画素描…

方向——到CALARTS，其实一直有点找不到方向——因为我知道我在训练一种工具，就像编程中的“API大师”，莫得灵魂！而最基础的我在做的训练——透视、形体、人体、人面、画画的软技能（自信、习惯、对自己的了解、对工具的掌握），是在“美术”这一行当的基本功，我了解到版画、油画（却有点不感兴趣了）、插画（illustration）、产品设计、建筑设计etc。而animation是一个点燃我的东西，真的就像是这学期重拾音乐一样…——当时对animation的狂热啊！
先讲讲我的想法

  1. animation不会局限画种，任你天马行空地画！局限你的只有你的想象力
  2. ani要学好多软件
  3. 当时那个暑假，我被ani感动得一塌糊涂，我盘算着如何学动画，如何写剧本，如何完成自己的一部精美作品！
  4. 做动画的手段太多啦

就和知乎上所谓“弹心情”（这个窝认为弹吉他好高好高地境界）一样，绘画里就是天马行空地画，也许在很多人看来就是日常，但是可能就是我现在给自己的局限，去画自己地心情、昨晚的梦、电影中我想要它发生的一个情节、一个虚构的小说人物、一段自己讲给自己自己故事、设计的椅子、比如我地透视理论、比如我脸上的痘痘、比如我地情绪、打印很多我的照片尝试眼镜
谓之！天马！行空！
这好像就是我说想说的“人与人差距”，人们所思所想、看到的世界、体会到的世界会天差地别
一种生活方式！ 比如随身带着速写本

素描——必先利其器

  1. 光线，不要在纸上太乱
  2. 素描本的——
    用本子画画，难免要把本子摊在桌子上画，可一旦你的眼睛不是垂直于画面
  3. 削铅笔！
    “这有点像上战场前对自己兵器进行打磨的一种心理状态，也像去参加一个朋友聚会前，好好把自己的鞋带系得漂漂亮亮的一种踏实，衣领袖口，褶皱检查，一切都准备好了，这个时候，你的心态也会放松很多，这样的状态画画，是能帮助你在画面中真实的流露自己的感受与审美的。”
    1. 削普通
    2. 削尖角
    3. 削更尖
    4. 削芯

基础之——线条练习

  1. 三：写字式，运腕式，运臂式
  2. 写字——小拇指固定，用于刻画小范围
  3. 运腕——8cm左右，小拇指固定，<笔尖，手腕，小指>三点固定
  4. 作用：
    1. 构成色块——调子 eg：渐变调子练习
    2. 起稿线

形画准

  1. 画得像，它只是一个画画入门技巧而已。

  2. 你完全可以轻松学会它，你学会了就学会了，也不代表什么

  3. 方法2

    1. 网格定型——当画幅太大
      这样的方法太限制自由了，太枯燥了

    2. 方法二：几何形概括

      1. 几何形概括，和网格法的区别很大。,追求神似 ————概括
      2. 几何形概括，是从整体出发，讲究的是把握大概特征，而不在乎100％精准。
      3. 在几何形起形时，我们通常会建议大家以方形（多边形）类来概括，为什么呢？
      4. 多边形概括后，比圆形概括特征感更强
      5. 你仍然要铭记，要从大画到小。

PPS：

  1. 如果老师给你点评到此，让你自己去练习，去「悟」，你一定是很懵的

  2. 带入几何体的思维

  3. 速写的本质，其实是快速概括物体的特征

  4. 我们习惯了把一切物体都看得很复杂，比如在写实的世界里，你需要看到准确的形，过渡自然的颜色，丰富细腻的质感，沉甸甸的体积感，真实的空间感等。

  5. 15年，觉得对人体还是很不了解的，会找各种网上的教程来画

  6. 拥有超强的「学习能力」，才能拥有真正的自信
